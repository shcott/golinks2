Welcome to Golinks!

Here's what you can do with Golinks. To get started, type any of the following into your address bar:

  • Create a golink to the current page: go /mark <bookmark name>
  • Visit a golink: go <bookmark name>
  • Edit a golink: go /edit <bookmark name>
  • Delete a golink: go /delete <bookmark name>
  • Search all Chrome bookmarks: go /search <anything>
  • View and manage all golinks: go /list
  • See this help message: go /help
