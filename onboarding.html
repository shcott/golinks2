How to use Golinks

Welcome to Golinks! To return to this page at any time, type /? into the Golinks launcher.

Golinks is a bookmarks manager that works like a personal URL shortener. It's perfect for bookmarking webpages that you use often, all without using a mouse.

Getting Started

If you have used Quicksilver, Alfred, Launchy, Wox, or Gnome Do, you'll feel right at home with Golinks' launcher interface. Otherwise, you might find this guide useful to show you how to get started.

Step 1: Open the launcher

The default shortcut to open the Golinks launcher is Ctrl+Shift+Space (on Macs, Cmd+Shift+Space. You can also click on the Golinks icon to the right of the address bar.

Step 2: Create your first golink

Type /mark my first golink, then press Enter. The /mark command bookmarks the current page. The Golinks launcher will close after you complete the action.

Step 3: Visit your first golink

Open a new tab and then open the Golinks launcher again. The launcher shows the most recently added and visited golinks when opened, so your new golink will appear there. Press Enter to visit it.

Once you collect many golinks, you can start typing the keyword for the link you want into the search bar to find it. When you press Enter, it will follow the highlighted link.

Step 4: Everything else

Open the Golinks launcher and type /. This will show you everything else you can do with Golinks.

Congrats! You now know how to mark and follow golinks! If you find Golinks useful, please be sure to leave a review on the Chrome web store page.

Advanced Features

Import links

To import your existing bookmarks in bulk, use the /list command to get to Chrome's bookmark manager and then drag and drop your bookmarks into the Golinks folder. Do not drag entire folders into the Golinks folder; Golinks will ignore them.

Open links in new tabs and windows

To open a golink in a new tab, hold down Ctrl (Cmd on Macs) when clicking the link or pressing Enter.

To open a golink in the background, hold down Ctrl and Shift.

To open a golink in a new window, hold down Shift.

Search your Chrome bookmarks

You can use the Golinks launcher to find any bookmark in Chrome. In the Golinks launcher, type /search and then any part of the name or URL of the bookmark you're looking for.

Navigate to open tabs

The Golinks launcher allows you to search your open tabs by title or URL using the /tab command.

Search browsing history

Use the /history command to find a page from your recent browsing history.

Change the launcher shortcut

You can change which keyboard shortcut opens the Golinks launcher in Chrome's extension configuration.

Use variable golinks

You can add to the end of a URL when you visit it by typing a / after your golink keyword. For example, if you bookmarked https://reddit.com/r/ as a golink using the keyword r, you can visit your favorite subreddits by typing r/funny or r/all into the Golinks launcher.

Another great site to do this with is GitHub. If you have GitHub bookmarked as gh, you could view the source code for this extension using gh/xsznix/golinks2.

Technical Details and Caveats

How Golinks stores its data

Golinks creates a special folder in your Chrome bookmarks to store golinks. It identifies this folder by its title, "Golinks". You may move this folder if you prefer to put it somewhere else, but if you rename it, Golinks will not find the golinks inside it. If you already had a folder called "Golinks" before you installed Golinks, Golinks will use that folder for golinks. If you uninstall Golinks, your golinks will not be deleted so that you can still access them. If you then reinstall Golinks, it will use the Golinks folder from your previous installation (unless you renamed or deleted it).

Use your Golinks everywhere

Your Golinks are available anywhere you are signed into Chrome and are syncing your bookmarks. To access your golinks on mobile Chrome, use Chrome's bookmark search and navigate to your Golinks folder. The golinks you most recently accessed on desktop will be at the top, and accessing golinks on mobile will not move them to the top of the folder like on desktop.
